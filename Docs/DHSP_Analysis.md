# Guide to Daftar Harga Satuan Pekerjaan (DHSP)
**Based on:** SE Bina Konstruksi No. 30 Tahun 2025  
**Source File:** AHSP CIPTA KARYA SE BINA KONSTRUKSI NO 30 TAHUN 2025 Rev 1.xlsx  

---

## Overview
This document summarizes the **Daftar Harga Satuan Pekerjaan (DHSP)** — the official 2025 reference for standard unit prices across public-works construction.  
It helps project teams, engineers, and developers quickly identify cost ranges and common items when building price-scraping logic or dashboards.

Each section contains:
- **Category name**
- **Example work items**
- **Typical unit**
- **Average price range (Rp)**
- **Remarks / revision notes**

## Notes
- Prices follow **SE Bina Konstruksi No. 30 Tahun 2025** guidance.  
- Actual values may vary by **region**, **accessibility**, and **project scale**.  
- DHSP 2025 is used for **budget estimation**, **tender benchmarking**, and **public works cost control**.  
- For scraping integration, each sheet in the Excel corresponds to one DHSP category (e.g., Beton, Kayu, Sanitair).
